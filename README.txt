Objet du projet:
- écrire un batch pour parcourir une liste de fichiers JAR d'un répertoire donné et les dézipper à la racine.

- mode opératoire:
batch dezip.bat : décompresse les fichiers *.JAR présents dans le répertoire courant
batch listJar.bat : prend en paramètre un répertoire, et trace dans le fichier "listeJar.txt" les fichiers *.JAR qui sont présents
